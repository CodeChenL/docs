Configuration can be done using `rsetup`:

1. Run `rsetup` in the terminal.

2. Use the arrow keys and the Enter key to go to `User Settings` -> `Configure auto login`.

3. Use **space** to select the type of session to be automatically logged in,
   and then use the Enter key to confirm.

:::info
The session types are the following:

gdm, sddm, lightdm: GUI desktop manager

serial-getty: Serial debugging terminal

getty: Display terminal
:::
